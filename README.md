# AI Lead Generation Bot

This project is a simple AI-powered system designed to help businesses automate lead generation and qualification.

## Features
- AI-generated outreach messages
- Lead qualification logic
- Simple chatbot integration
- Customizable prompts

## Use Case
Agencies and businesses can use this tool to automate their sales pipeline.

## Tech Stack
- Python
- OpenAI API
- Zapier (optional)

## Future Plans
- WhatsApp integration
- CRM automation
- Multi-language support
